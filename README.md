# Url Validation


<h6>Author : RKT</h6>


url-validation is a  Valid Url(Universal Resource Locator) Detecting Tool.


### Setup ###

+ sudo pip3 install -r requirements.txt

### Terminal Command ###

+ git clone https://github.com/r3k4t/url-validation.git
+ cd url-validation
+ sudo python3 url-validation.py
